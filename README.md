***Project Overview***

This project presents an interactive Sales Performance Dashboard built using Power BI to analyze revenue, profitability, sales performance, and customer behavior. The dashboard transforms raw sales data into actionable insights that support data driven decision making.

**Tools I Used:**
- Power BI – Dashboard development and visualization
- Power Query – Data cleaning and transformation
- DAX (Data Analysis Expressions) – KPI and measure calculations
- GitHub – Project documentation and version control

**Dateset:** Sales dataset (Kaggle-style dataset)
Records: ~1,000 transactions.
Key fields: Sales, Profit, Quantity, Product Category, Region & Sales Representative, Customer Type (New vs Returning), Payment Method, Sales Channel (Online vs Retail), Date

**Key Insights:**
Sales performance is relatively balanced across regions, with slight concentration in top-performing areas, suggesting opportunities to expand growth in underperforming regions. Sales representatives demonstrate consistent performance with minimal gaps between top and lowest performers, indicating a stable and evenly distributed sales team. Customer purchasing behavior is evenly split between new and returning customers, reflecting strong acquisition and retention efforts, while retail slightly outperforms online sales. Additionally, digital payment methods such as credit cards and bank transfers dominate transactions, highlighting the importance of convenient, non-cash payment options in driving overall sales.

In order to use the interactive dashboard: Download the .pbix file, Open Power BI Desktop, interact with the filters on the dashboard
